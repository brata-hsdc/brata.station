Raspbian installation.

Set up to run custom application on start-up:

   apt-get install upstart
   created /etc/init/sve.conf

Redirect logs to central log server:

   modified /etc/rsyslog.conf

Move /var to RAM to avoid filling up SD card and to avoid writes to SD
card:

   modified /etc/fstab
   mv /var /var.old
   mkdir /var
</strike>


Clean-up Install Archives:

   $ sudo apt-get clean

Install Python Requests

   $ sudo apt-get install python-requests

Install Wiremock

   $ sudo mkdir /opt/wiremock-1.48
   $ sudo chown pi:pi /opt/wiremock-1.48
   $ sudo ln -s wiremock-1.48 /opt/wiremock
   $ cd /opt/wiremock
   $ wget 'http://repo1.maven.org/maven2/com/github/tomakehurst/wiremock/1.48/wiremock-1.48-standalone.jar'
   $ chmod 444 wiremock-1.48-standalone.jar

Notes for MS/HMB ICD:
   Change connect example: "connect_hmb" to "connect".


Install Flask

   $ sudo apt-get install python-flask
